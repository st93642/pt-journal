# Certified Ethical Hacker (CEH) v12 - Quiz Content

This directory contains quiz questions for the CEH v12 certification exam preparation, organized by the 20 official EC-Council modules.

## CEH v12 Module Structure

The CEH v12 certification covers 20 comprehensive modules:

1. **Introduction to Ethical Hacking** - Ethics, methodologies, frameworks, laws
2. **Footprinting and Reconnaissance** - Passive/active recon, OSINT, search engines
3. **Scanning Networks** - Port scanning, network discovery, OS fingerprinting
4. **Enumeration** - Service enumeration, NetBIOS, SNMP, LDAP, NFS
5. **Vulnerability Analysis** - Vulnerability scanning, assessment, classification
6. **System Hacking** - Password cracking, privilege escalation, covering tracks
7. **Malware Threats** - Trojans, viruses, worms, ransomware, APTs
8. **Sniffing** - Packet capture, ARP poisoning, MAC flooding, MITM
9. **Social Engineering** - Phishing, pretexting, baiting, tailgating
10. **Denial of Service** - DoS/DDoS attacks, botnets, detection, mitigation
11. **Session Hijacking** - Session management, hijacking techniques, prevention
12. **Evading IDSs, Firewalls, and Honeypots** - IDS/IPS evasion, firewall bypass
13. **Hacking Web Servers** - Web server attacks, misconfigurations, patches
14. **Hacking Web Applications** - OWASP Top 10, injection, XSS, CSRF
15. **SQL Injection** - SQL attacks, blind SQLi, automated tools, mitigation
16. **Hacking Wireless Networks** - WEP/WPA/WPA2/WPA3, wireless attacks, tools
17. **Hacking Mobile Platforms** - Android/iOS attacks, MDM, mobile malware
18. **IoT and OT Hacking** - IoT vulnerabilities, SCADA, ICS, smart devices
19. **Cloud Computing** - Cloud security, container attacks, serverless, AWS/Azure
20. **Cryptography** - Encryption algorithms, PKI, SSL/TLS, hashing, digital signatures

## Question File Format

Each module contains a `.txt` file with 50 multiple-choice questions in the following format:

```
Q: What is the primary purpose of ethical hacking?
A) To exploit vulnerabilities for financial gain
B) To identify and fix security weaknesses
C) To create malware
D) To bypass security controls
CORRECT: B

Q: Which phase comes first in the ethical hacking methodology?
A) Scanning
B) Reconnaissance
C) Gaining Access
D) Maintaining Access
CORRECT: B
```
